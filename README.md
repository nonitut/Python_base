# Python

| **Характеристика**     | **Кортеж (tuple)**                   | **Список (list)**                     |
|-------------------------|---------------------------------------|---------------------------------------|
| **Изменяемость**        | Неизменяемый (immutable)             | Изменяемый (mutable)                  |
| **Синтаксис**           | Записывается в круглых скобках `()`  | Записывается в квадратных скобках `[]` |
| **Скорость работы**     | Быстрее, так как неизменяемый        | Медленнее, так как изменяемый         |
| **Использование памяти**| Требует меньше памяти                | Требует больше памяти                 |
| **Основное назначение** | Для хранения неизменяемых данных     | Для хранения изменяемых данных        |
| **Методы**              | Имеет меньше встроенных методов      | Имеет больше встроенных методов       |
